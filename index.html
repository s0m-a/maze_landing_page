the maze

a 3d maze game using SDL2 and Raycasting

view source code

FEATURE

In our implementation of the maze game, we employed ray-casting to construct the walls, drawing inspiration from LodeV's method of using vectors to determine ray length. Maps are represented in 2D arrays within text files, which are then parsed as arguments to the maze executable. Within these arrays, 0 signifies open space, while any other integers are depicted as walls. To render this we utilized the C language alongside the SDL2 library.For navigation within the maze, we established the following fundamental controls:
W: Move forward
S: Move backward
Q: Move left
E: Move right
ESC: Quit maze
